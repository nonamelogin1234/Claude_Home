# CLAUDE CODE — ТОЧКА ВХОДА

> Читается автоматически при старте каждой сессии.

## Старт
1. `git pull`
2. Прочитай `brain/` (@brain/infra.md, @brain/grabli.md, @brain/projects.md, @brain/rules.md)
3. Если работаем над проектом — прочитай его `context.md`
4. Напиши одной строкой: «Прочитано: brain/ + [проект]»

## Конец сессии
По команде «пуш» или «конец сессии»:
1. Обнови файлы в `brain/` и папке проекта
2. `git add . && git commit -m "описание" && git push`

## Правила
- Пути всегда относительные
- Грабли только дополнять, никогда не удалять
- После каждой задачи — коммит
- Язык общения: русский
