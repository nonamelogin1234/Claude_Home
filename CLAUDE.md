# CLAUDE CODE — ТОЧКА ВХОДА

> Читается автоматически при старте каждой сессии Claude Code.
> Полные знания — в brain/*.md. Читай по запросу, ищи через `python brain/search.py`.

## Ядро (загружается автоматически)

@brain/core.md

`core.md` определяет компьютер (рабочий/домашний/homeserver/VPS) и говорит, какой файл `brain/core-work.md` или `brain/core-home.md` читать дальше — прочитай его перед началом работы.

## Старт сессии

1. `/start` (skill) или вручную: `git pull`
2. Прочитать `brain/core.md` → определить окружение → прочитать соответствующий `brain/core-work.md` / `brain/core-home.md`
3. Если работаем над проектом — прочти его `context.md`
4. Одна строка подтверждения: «Прочитано: core-work/core-home + [проект если есть]»

## Конец сессии

По команде «пуш», «конец сессии» или «новый чат» → `/finish` (skill):
1. Обнови `brain/projects.md` и `context.md` проекта
2. `git add -A && git commit -m "описание на русском" && git push`

## Авто-пуш

После каждого логически завершённого коммита **сразу делать `git push`** без ожидания команды.
Логически завершённый момент — когда задача выполнена, файлы сохранены, коммит сделан.
Не пушить только если: работа явно не закончена, или пользователь сказал «не пушь».
