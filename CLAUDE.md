# CLAUDE CODE — ТОЧКА ВХОДА

> Читается автоматически при старте каждой сессии Claude Code.
> Полные знания — в brain/*.md. Читай по запросу, ищи через `python brain/search.py`.

## Ядро (загружается автоматически)

@brain/core.md

## Старт сессии

1. `/start` (skill) или вручную: `git pull`
2. Если работаем над проектом — прочти его `context.md`
3. Одна строка подтверждения: «Прочитано: core + [проект если есть]»

## Конец сессии

По команде «пуш», «конец сессии» или «новый чат» → `/finish` (skill):
1. Обнови `brain/projects.md` и `context.md` проекта
2. `git add -A && git commit -m "описание на русском" && git push`
