---
description: Добавить/настроить MCP-сервер в конфиге Claude Code или Claude Desktop
---

Задача: добавить MCP-сервер `$ARGUMENTS` (или то, что описал пользователь) в правильный конфиг на этом компьютере.

## 1. Определи, каким приложением пользуется пользователь на этом ПК

- **Claude Code (CLI)** → конфиг `.mcp.json` в корне репозитория (`C:\Users\torganov-a\Documents\Claude_Home\.mcp.json`), коммитится в git, синкается на все компьютеры автоматически.
- **Claude Desktop** → отдельный локальный файл, **НЕ в git**, путь зависит от того, как приложение установлено:
  - Обычная установка: `%APPDATA%\Claude\claude_desktop_config.json`
  - MSIX/Windows Store (виртуализованная ФС): `%LOCALAPPDATA%\Packages\Claude_<hash>\LocalCache\Roaming\Claude\claude_desktop_config.json`
  - Проверь оба пути — если оба существуют, сначала сравни (может быть junction/один и тот же файл — правка одного могла отразиться в другом).
  - См. `brain/workstations.md` → таблицу "MCP-серверы Claude Desktop" для актуальных путей по каждому компьютеру.

Если непонятно, каким клиентом пользуется человек за этим ПК — посмотри `brain/core-work.md` / `brain/core-home.md`, там обычно зафиксировано.

## 2. Перед правкой

1. Прочитать текущий конфиг целиком.
2. Сделать бэкап (`<файл>.backup`).
3. Никогда не перезаписывать файл целиком — добавлять новый ключ в существующий объект `mcpServers`, не трогая остальные серверы и `preferences`.

## 3. Правка

Добавь блок вида:
```json
"<имя-сервера>": {
  "command": "<путь или npx>",
  "args": ["-y", "<пакет>"],
  "env": { "KEY": "value" }
}
```
- Секреты (токены, ключи) — не хардкодить в открытом виде в репо-файлы, для `claude_desktop_config.json` это локальный файл вне git, туда можно; для `.mcp.json` — только через переменные окружения или ссылку на Vaultwarden.
- Если сервер требует локальной установки (npm/uv/pip пакет, бинарник) — сначала установи его, пропиши абсолютный путь к исполняемому файлу, если `npx`/`uvx` неудобны или недоступны офлайн.

## 4. Валидация

Провалидируй итоговый JSON на синтаксическую корректность.

## 5. После сохранения

- Для Claude Desktop — напомни пользователю **полностью** закрыть приложение (включая трей) и открыть заново.
- Для Claude Code — новый MCP из `.mcp.json` подхватывается при следующем старте сессии/`claude mcp list`.
- Зафиксируй результат в `brain/workstations.md` (таблица MCP по компьютерам) — статус 🟢/🟡, что установлено, что осталось сделать руками (например OAuth-логин).
