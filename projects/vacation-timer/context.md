# ТАЙМЕР ДО ОТПУСКА

## ЦЕЛЬ
Настольный виджет Windows, который показывает, сколько рабочих часов осталось до отпуска.
Считает только будние дни с 09:00 до 17:00 за период с 18.05.2026 по 29.05.2026.

## СТАТУС
Готово - виджет установлен и запущен на рабочем компьютере через ярлык `До отпуска`.

## СЛЕДУЮЩИЙ ШАГ
- Не требуется; при желании можно позже добавить запуск вместе с входом в Windows.

## АРХИТЕКТУРА
- `vacation-timer.ps1` - окно виджета, оформление и расчёт прогресса.
- `launch-vacation-timer.vbs` - тихий запуск без консольного окна.
- `install-shortcut.ps1` - создание ярлыка на рабочем столе текущего пользователя.
- `assets/design-concept.png` - визуальная концепция, по которой собран интерфейс.

## РАСЧЁТ
- Начало отсчёта: `18.05.2026 09:00:00`.
- Конец отсчёта: `29.05.2026 17:00:00`.
- Общее время: `80:00:00`, то есть 10 рабочих дней по 8 часов.
- Вечером, ночью и в выходные показания не увеличиваются.

## ТЕХНИЧЕСКИЕ ДЕТАЛИ
```powershell
# Запуск виджета
wscript.exe projects/vacation-timer/launch-vacation-timer.vbs

# Установка ярлыка на текущий рабочий стол
powershell -NoProfile -ExecutionPolicy Bypass -File projects/vacation-timer/install-shortcut.ps1

# Проверка расчёта для известного времени
powershell -NoProfile -ExecutionPolicy Bypass -File projects/vacation-timer/vacation-timer.ps1 -SelfTest -TestNow "2026-05-25 10:28:18"
```
