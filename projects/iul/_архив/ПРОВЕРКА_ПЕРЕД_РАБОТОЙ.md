# Быстрая проверка перед продолжением

1. Проверить синтаксис:

```powershell
python -m py_compile iul_prototype.py
```

2. Запустить прототип:

```powershell
.\Запустить_прототип.bat
```

3. Протестировать 2 файла в одной пачке:

```text
Раздел ПД  № 3 АР часть №4.pdf
Раздел ПД № 5 подраздел ПД № 1 ИОС1 часть № 1 книга № 3.pdf
```

4. Проверить результаты:
- PDF создан в выбранной папке.
- Excel создан в `ИУЛ_в_ред_формате`.
- Рядом появился `ИУЛ_журнал_*.txt`.
- В журнале нет лишнего предупреждения про два номера раздела.
- В `D5`: `Часть 4`, `Книга 3`, с пробелами.
- В датах используется `[Дата_последнего_изменения]`.

5. Если GUI зависает:
- проверить worker-thread и очередь `self.ui_queue`;
- проверить, что все обращения к Tkinter идут только в главном потоке через `process_ui_queue`;
- проверить Excel COM в потоке: `pythoncom.CoInitialize()` вызывается внутри `ExcelSession.__enter__`.
