# ИУЛ — Информационно-удостоверяющий лист

Программа генерирует ИУЛ для проектной документации:  
вход — PDF-файлы, выход — заполненный Excel (.xlsx) + PDF на каждый файл.

---

## Версии проекта

### v1 — Tkinter ← **ПРИОРИТЕТ, АКТИВНАЯ РАЗРАБОТКА**

Папка: `v1_tkinter/`  
Паспорт: [`v1_tkinter/ПАСПОРТ_v1.md`](v1_tkinter/ПАСПОРТ_v1.md)

```
v1_tkinter/
├── iul_prototype.py     ← вся логика + Tkinter UI
├── ИУЛ.spec             ← сборка exe
├── config.json          ← данные проекта
├── ИУЛ_шаблон.xlsx      ← Excel-шаблон
├── Подписи/             ← PNG-подписи
└── ПАСПОРТ_v1.md        ← полный паспорт v1
```

**Запуск:**
```powershell
cd "...\Claude_Code\v1_tkinter"
python iul_prototype.py
```

**Сборка exe:**
```powershell
cd "...\Claude_Code\v1_tkinter"
python -m PyInstaller ИУЛ.spec --clean
# → v1_tkinter\dist\ИУЛ.exe
```
