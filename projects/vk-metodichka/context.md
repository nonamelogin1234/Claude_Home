## Ключевые технические решения

- PP-R для В1/ГВС, **тихий PP** (Wavin Optima / Geberit Silent-db20) для К1 у классов
- Т3 ≥ 60°C постоянно (ночной режим — снижать обороты насоса, но не температуру)
- Термостатические смесители 38–42°C в детских зонах (Oras 6161, HANSA Vantis)
- В2 — только сталь, кольцевая схема при >12 ПК, редукторы ≤0,40 МПа
- Вероятностный метод: N=132, p=0,107, q(В1)=4,7 л/с; мощность ИТП≈450 кВт
