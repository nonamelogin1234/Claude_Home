### Система еженедельного скоринга

Счёт ведётся по шести блокам, баллы от отрицательных до положительных, пола нет:

| Блок | Критерии | Баллы |
|------|---------|-------|
| Питание (каждый день) | ≤лимита / нейтрально / выше +500 | +2 / 0 / −2 |
| Силовые | 3/3 / 2/3 / 1/3 | +6 / +3 / +1 |
| Сон | ≥7.5ч / 6.5-7.4ч / <6.5ч | +4 / 0 / −2 |
| Шаги | ≥8000 / 4000-7999 / <4000 | +3 / 0 / −1 |
| Кардио | есть / нет | +2 / 0 |
| Алкоголь | неделя без / был | +2 / 0 |

Лимиты калорий: будни ≤2 500 / пятница ≤2 700 / сб-вс ≤2 800
