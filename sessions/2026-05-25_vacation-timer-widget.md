# СЕССИЯ: ВИДЖЕТ ДО ОТПУСКА

## РЕЗУЛЬТАТ
Создан и установлен локальный настольный виджет Windows `До отпуска` для отсчёта рабочих часов до отпуска.

## ЧТО СДЕЛАНО
- Создан проект `projects/vacation-timer/`.
- Реализован WPF-виджет в тёмной теме с круговым и линейным прогрессом.
- Настроен подсчёт только по будням в интервале `09:00-17:00`.
- Период отсчёта: с `18.05.2026 09:00:00` до `29.05.2026 17:00:00`, всего `80:00:00`.
- Добавлен тихий запуск без консольного окна и ярлык рабочего стола `До отпуска`.
- Исправлено перетаскивание окна за свободную область панели.
- По просьбе пользователя виджет уменьшен вдвое до размера `215 x 300`.

## ПРОВЕРЕНО
- В `25.05.2026 10:28:18` расчёт выдаёт: прошло `41:28:18`, осталось `38:31:42`, прогресс `51.8%`.
- В выходные и после `17:00` счётчик стоит на паузе.
- В `29.05.2026 17:00:00` прогресс достигает `80:00:00`.
- Окно визуально проверено на рабочем столе; ярлык запускает приложение; компактное окно перетаскивается.
