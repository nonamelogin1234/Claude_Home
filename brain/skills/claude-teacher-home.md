## Подключение к серверам

### Shell API — запуск команд на VPS (для Клода)

```powershell
powershell -ExecutionPolicy Bypass -File C:\Users\user\srv.ps1 -cmd "КОМАНДА"
```

Альтернатива — напрямую через API:
```
POST https://mcp.myserver-ai.ru:7723
X-Secret: shell-api-secret-2026
{"cmd": "команда"}
```

### Shell API — запуск команд на домашнем сервере (для Клода)

```powershell
powershell -ExecutionPolicy Bypass -File C:\Users\user\home.ps1 -cmd "КОМАНДА"
```

Маршрут: home.ps1 → nginx VPS :7724 → домашний сервер :7722

### SSH для Сергея — терминал на серверах

**VPS:**
```bash
ssh sergei@147.45.238.120
```

**Домашний сервер (прямое подключение по локальной сети):**
```bash
ssh sergei@192.168.0.106
```

> ℹ️ С домашнего ПК подключаться напрямую по IP — VPS не нужен.
> ⚠️ SSH не работает при включённом WireGuard VPN на домашнем ПК — отключи WireGuard перед SSH.

## Железные правила (не меняются никогда)

1. **Не логировать сессию без явного разрешения Сергея.** Спросить в конце.
2. **Сергей вводит команды сам.** Ты показываешь команду — он вводит. Не писать «скопируй и выполни».
3. **Не давать готовый ответ когда ученик близок к выводу.** Задать вопрос вместо ответа.
