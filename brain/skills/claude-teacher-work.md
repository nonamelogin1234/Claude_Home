## Подключение к серверам

### Клод запускает команды на VPS

```powershell
powershell -ExecutionPolicy Bypass -File C:\Users\torganov-a\srv.ps1 -cmd "КОМАНДА"
```

Альтернатива — напрямую через API:
```
POST https://mcp.myserver-ai.ru:7723
X-Secret: shell-api-secret-2026
{"cmd": "команда"}
```

### Клод запускает команды на домашнем сервере

Через VPS по SSH — работает без пароля (ключ настроен в апреле 2026):

```powershell
powershell -ExecutionPolicy Bypass -File C:\Users\torganov-a\srv.ps1 -cmd "ssh -o StrictHostKeyChecking=no sergei@10.8.0.27 'КОМАНДА'"
```

### SSH для Сергея — интерактивный терминал

**VPS** (без пароля, ключ добавлен):
```bash
ssh sergei@147.45.238.120
```

**Домашний сервер через ProxyJump** (без пароля):
```bash
ssh -J sergei@147.45.238.120 sergei@10.8.0.27
```

**Домашний сервер в два шага:**
```bash
ssh sergei@147.45.238.120   # сначала на VPS
ssh sergei@10.8.0.27        # потом на домашний
```

> ℹ️ Порт 7724 — это shell API домашнего сервера, НЕ SSH.
> ℹ️ nginx на VPS без модуля stream — прямое TCP-проксирование SSH не работает.

## Железные правила (не меняются никогда)

1. **Не логировать сессию без явного разрешения Сергея.** Спросить в конце.
2. **Сергей вводит команды сам.** Показывай команду — он вводит. Не писать «скопируй и выполни».
3. **Не давать готовый ответ когда ученик близок к выводу.** Задать вопрос вместо ответа.
