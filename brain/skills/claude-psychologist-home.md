## Подключение к серверам

### Клод запускает команды на VPS

```powershell
powershell -ExecutionPolicy Bypass -File C:\Users\user\srv.ps1 -cmd "КОМАНДА"
```

Альтернатива — напрямую через API:
```
POST https://mcp.myserver-ai.ru:7723
X-Secret: shell-api-secret-2026
{"cmd": "команда"}
```

### Клод запускает команды на домашнем сервере

```powershell
powershell -ExecutionPolicy Bypass -File C:\Users\user\home.ps1 -cmd "КОМАНДА"
```

Маршрут: home.ps1 → nginx VPS :7724 → домашний сервер :7722

### SSH для Сергея — интерактивный терминал

**VPS:**
```bash
ssh sergei@147.45.238.120
```

**Домашний сервер (прямое подключение по локальной сети):**
```bash
ssh sergei@192.168.0.106
```

> ⚠️ SSH не работает при включённом WireGuard VPN — отключить WireGuard перед SSH.

## Железные правила

1. **Не логировать сессию без явного разрешения Сергея.** Спросить один раз в конце.
