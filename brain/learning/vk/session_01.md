# Сессия 01 — Основы ВК + разбор проекта школы

## Дата
Май 2026

## Что прошли

### 1. Исходные данные для проектирования ВК
- Начало любого проекта: ТЗ → ТУ
- ТУ от Водоканала: на водоснабжение и канализацию (два отдельных документа)
- ТУ от Теплосети: если ГВС централизованное
- ТУ от Горгаза: если своя котельная/ИТП на газе
- Ключевой параметр из ТУ Водоканала — **гарантируемый свободный напор** в точке подключения

### 2. Гидравлика — базовая логика
- 1 атм = 10 м.вод.ст.
- Подъём воды на 1 метр = 0,1 атм потерь
- Минимальный напор у прибора (смеситель, умывальник) = 0,2 атм (2 м.вод.ст.) — по СП 30.13330
- **Формула баланса:** Напор на вводе − потери на высоту − потери в трубах ≥ минимальный напор у прибора
- Если не хватает → насосная станция (ПНС)
- Насос ставят даже при достаточном давлении: нестабильность городской сети, сложный объект с несколькими блоками
- Насос с частотным регулированием — поддерживает стабильное давление

### 3. Структура системы ВК в здании
```
Ввод → Водомерный узел → Магістраль (подвал) → Стояки → Подводки → Приборы
```
- **Магістраль** — горизонтальная труба в подвале, большой диаметр
- **Стояк** — вертикальная труба, один стояк = одна «колонка» санузлов на всех этажах
- **Подводка** — короткая труба на этаже от стояка до прибора
- Санузлы в жилых домах проектируют друг над другом — один стояк на все этажи

### 4. Запорная арматура у основания стояка
- Два крана: верхний перекрывает подачу, нижний — дренажный (слив воды из стояка)
- Логика: закрыл верхний (отсёк от магістрали) → открыл нижний → вода из стояка слилась → ремонт без отключения всего здания

### 5. Гидравлический расчёт — первый опыт
Реальные данные из проекта школы (Блок А, 3 этажа):
- Напор на вводе: 26 м.вод.ст. = 2,6 атм
- Высота подъёма до 3-го этажа: ~10,8 м = 1,08 атм
- Минимальный напор у прибора: 0,2 атм
- Потери в подводке Ø20, 2 м: 0,06 атм (i = 0,03 атм/м)
- **Итого нужно:** 1,08 + 0,2 + 0,06 = **1,34 атм**
- **Запас:** 2,6 − 1,34 = 1,26 атм — давления хватает
- Насос в школе всё равно есть (в Блоке Б) — из-за 5 блоков и бассейна

### 6. Принцип диктующей точки
- Расчёт ведётся по **самому невыгодному прибору** (самый дальний и высокий)
- Если для него хватает давления — для всех остальных тоже хватит

### 7. Диаметры труб
- Магістраль Ø50 → Стояк Ø25 → Подводка Ø20
- Диаметры уменьшаются по мере уменьшения расхода
- Узкая труба → вода быстрее → потери больше → поэтому на магістрали диаметр больше
