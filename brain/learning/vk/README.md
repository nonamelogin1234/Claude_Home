# Обучение ВК (водоснабжение и канализация)

## Цель
Научиться самостоятельно проектировать раздел ВК жилого здания — от получения ТУ до выпуска комплекта чертежей.

## Контекст
- Фон: инженер, помощник ГИПа, работает в проектной организации
- Опыт в ВК: нулевой
- Доступ к реальным проектам из архива организации
- Метод обучения: разбор реальных готовых проектов + параллельное создание своего

## Учебный объект
Жилой дом 14 этажей, 6 квартир на этаже (студии, 1-комнатные, до 4-комнатных), есть подвал.
АР (РД) + ГП из ПОС — взяты из архива организации.
ТУ от Водоканала — придумываем сами по аналогии с реальными.

## Метод работы с Клодом
1. В начале сессии: «иди в brain/learning/vk, почитай контекст, продолжаем ВК»
2. Клод читает README.md + последний session_XX.md
3. Продолжаем с того места где остановились
