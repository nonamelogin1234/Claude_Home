# ИНИЦИАЛИЗАЦИЯ ДЛЯ CLAUDE CHAT

> Этот файл — инструкция для claude.ai (не для Claude Code).
> Claude Code читает CLAUDE.md в корне — этот файл для него не нужен.

## Когда читать
Только по явной команде: «читай контекст» / «загрузи контекст» / «инициализируйся».
По умолчанию в каждом чате НЕ читать ничего из репо.

## Что читать (в таком порядке)

| Файл | Зачем | Токены |
|------|-------|--------|
| `brain/projects.md` | Статусы всех проектов — читать всегда | ~200 |
| `brain/infra.md` | Серверы, порты, подключения — читать всегда | ~400 |
| `brain/grabli.md` | Что не делать — читать всегда | ~300 |
| `brain/rules.md` | Правила ведения файлов — читать всегда | ~200 |

**Итого brain/: ~1100 токенов. Это норм.**

## Что НЕ читать по умолчанию
- `docai/` — там код (~3000+ токенов), читать только если работаем над DocAI
- `kinoclaude/` — читать только если работаем над КиноКлод
- `sessions/` — читать только если явно попросили найти что-то из прошлых сессий

## Если работаем над конкретным проектом
Дополнительно прочитать его `context.md`:
- DocAI → `docai/context.md`
- КиноКлод → `kinoclaude/context.md`

## После прочтения
Написать одной строкой:
«Прочитано: brain/ [+ название проекта если читался]»

## Как записывать саммари сессий
Если сессия была важной — создать файл `sessions/ГГГГ-ММ-ДД_тема.md`
Структура:
```
# Тема — Дата
## Что обсуждали
## Что решили
## Что сделали
## Следующий шаг
```
