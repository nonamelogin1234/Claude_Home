# ИНИЦИАЛИЗАЦИЯ ДЛЯ CLAUDE CHAT

> Этот файл — инструкция для claude.ai (не для Claude Code).
> Claude Code читает CLAUDE.md в корне — этот файл для него не нужен.

## Когда читать
Только по явной команде: «читай контекст» / «загрузи контекст» / «инициализируйся».
По умолчанию в каждом чате НЕ читать ничего из репо.

## Что читать (в таком порядке)

| Файл | Зачем | Токены |
|------|-------|--------|
| `brain/projects.md` | Статусы всех проектов — читать всегда | ~300 |
| `brain/infra.md` | Серверы, порты, подключения — читать всегда | ~500 |
| `brain/grabli.md` | Что не делать — читать всегда | ~400 |
| `brain/rules.md` | Структура репо и правила — читать всегда | ~300 |
| `brain/workstations.md` | MCP и локальные инструменты по компьютерам — читать при переносе настроек | ~500 |

**Итого brain/: ~1500 токенов. Норм.**

## Что НЕ читать по умолчанию
- Папки `projects/` — там код, читать только при работе над конкретным проектом
- Папки `infra/` — читать только при работе с инфраструктурой
- `sessions/` — только если просят найти что-то из прошлого
- `brain/coding.md` — только при написании кода

## Если работаем над конкретным проектом
Дополнительно прочитать его `context.md`:
- КиноКлод → `projects/kinoclaude/context.md`
- DocAI → `projects/docai/context.md`
- RPG Трекер → `projects/rpg-tracker/context.md`
- Revit MCP → `projects/revit-mcp/context.md`
- Homepage → `infra/homepage/context.md`
- И т.д. по структуре

## После прочтения
Написать одной строкой:
«Прочитано: brain/ [+ название проекта если читался]»

## Как записывать саммари сессий
Если сессия была важной — создать файл `sessions/ГГГГ-ММ-ДД_тема.md`
Структура:
```
# Тема — Дата
## Что обсуждали
## Что решили
## Что сделали
## Следующий шаг
```

## Команда «Переходим в новый чат»
Если пользователь пишет «Переходим в новый чат», нужно:
1. Зафиксировать текущий контекст в `context.md`, `brain/projects.md`, `brain/memory-graph.jsonl` и при необходимости `brain/workstations.md`.
2. Создать саммари в `sessions/`.
3. Сделать commit + push.
4. Выдать готовый текст для следующего чата.
