# DocAI v2 — следующий шаг: RAG поверх JSON

## Контекст

docai.py готов и работает. JSON с текстом и числами есть.
Проблема: документация большая, в контекст LLM целиком не влезает.

## Что нужно сделать

Построить локальный RAG поверх того что уже есть:

1. Читать JSON от docai.py
2. Разбивать текст на чанки
3. Генерировать embeddings локально
4. Хранить локально (никакого VPS, никакого сервера)
5. По вопросу пользователя — находить релевантные чанки и отвечать

## Интерфейс

Минимальный CLI. Пример:
    python ask.py "какая высота этажа?"
    python ask.py "где упоминается отметка -1.200?"

## Ограничения

- Всё локально, Windows, без серверов
- Документация конфиденциальная — никаких внешних API кроме Anthropic
  (Anthropic API key уже есть в окружении)
- Чертёжные страницы читаются плохо — известная проблема, не трогать
