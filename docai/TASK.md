# DocAI — PDF в JSON для проверки проектной документации

## Задача

Нужен локальный инструмент на Python, который принимает PDF файл проектной документации (строительство, архитектура) и возвращает структурированный JSON со всем текстом и числами с контекстом.

Этот JSON потом вручную загружается в Claude для анализа — например:
- «найди все места где площадь дома указана по-разному»
- «проверь, везде ли одинаковые отметки этажей»
- «найди противоречия в спецификациях»

## Характер документов

Российская проектная документация (кириллица). Форматы:
- Текстовые листы (пояснительные записки, спецификации)
- Чертёжные листы — сканы или векторные, с таблицами, размерными цепочками, отметками
- Таблицы спецификаций (позиция, наименование, количество, масса и т.д.)
- Смешанные листы

Числа в документах: площади, отметки (±0.000, -1.200), размеры (мм, м), нагрузки, количества, марки материалов.

## Известные проблемы — реальный опыт

Это не предположения, а то что уже проверено на реальных PDF:

**PyMuPDF и кириллица:**
- Проблема: Identity encoding. Текст извлекается как мусор (кракозябры).
- Коварство: `is_garbled()` не помогает — битый текст выглядит как валидный Unicode, функция не срабатывает.
- То есть PyMuPDF молча возвращает мусор без каких-либо ошибок.

**pdfminer.six:**
- Предположительно лучше справляется с Identity encoding, но на этих конкретных PDF не проверялось.

**tesseract:**
- Работает, русский язык (`tesseract-ocr-rus`) установлен.
- Использовали как fallback для нечитаемых страниц.

**Общий вывод:**
Главная задача — надёжно извлечь кириллический текст. Если какая-то библиотека даёт мусор вместо текста, нужно это детектировать и переключаться на другой метод.

## Требования к результату

На выходе должен быть JSON файл (не БД, не сервер — просто файл на диске).
Содержимое JSON — на твоё усмотрение, главное чтобы Claude потом мог по нему работать.
Инструмент должен запускаться на Windows без сервера, без интернета.

## Финальная цель

По возможности упаковать в .exe через PyInstaller чтобы можно было запускать без Python.
